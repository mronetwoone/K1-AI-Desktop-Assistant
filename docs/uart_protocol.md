# 串口通信协议

## K1 -> ASRPRO

用于姿态和环境播报。K1 发送后，ASRPRO 根据命令码播放本地语音片段。

| 数据包 | 含义 |
|---|---|
| `AA 01 55 0D 0A` | 坐姿恢复正常 |
| `AA 02 55 0D 0A` | 轻度驼背提醒 |
| `AA 03 55 0D 0A` | 轻度高低肩提醒 |
| `AA 04 55 0D 0A` | 明显驼背提醒 |
| `AA 05 55 0D 0A` | 明显高低肩提醒 |
| `AA 06 55 0D 0A` | 背部和肩部同时异常 |
| `AA 07 55 0D 0A` | 画面无人，可选 |
| `AA 20 温度 湿度 55 0D 0A` | 温湿度播报 |
| `AA 21 55 0D 0A` | 温湿度读取失败 |

## ASRPRO -> K1

| 数据包 | 含义 |
|---|---|
| `AB 20 55` | 请求 K1 读取 STM32 表情板上的环境传感器数据 |

## K1 -> STM32 表情屏

文本命令，波特率 115200，8N1，命令以 CRLF 结尾。

常用命令：`face`、`sensor`、`sensoron`、`sensoroff`、`loving`、`happy`、`sad`、`thinking`、`surprised`、`sleepy`、`angry`、`neutral`。
