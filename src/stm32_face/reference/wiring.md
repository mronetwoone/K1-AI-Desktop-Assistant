# STM32 传感器接线

## 引脚分配总表

| 传感器 | 信号 | STM32 引脚 | 备注 |
|--------|------|------------|------|
| **OLED SSD1309** | SCK | PA0 | 软件 SPI 时钟 |
| | SDA (MOSI) | PA1 | 软件 SPI 数据 |
| | CS | PA2 | 片选 |
| | DC | PA3 | 数据/命令 |
| | RST | PA4 | 复位 |
| **DHT11** | DATA | **PB0** | 单总线，需外部 10kΩ 上拉到 VCC |
| **MQ-2** | AOUT | **PA6** | ADC1_CH6，模拟输出 |
| | DOUT | **PB5** | 数字输出（模块比较器） |
| **BH1750** | SCL | **PB6** | I2C1 时钟，需 4.7kΩ 上拉到 3.3V |
| | SDA | **PB7** | I2C1 数据，需 4.7kΩ 上拉到 3.3V |
| **UART1** | TX | PA9 | 接 CH340/CP2102 的 RX |
| | RX | PA10 | 接 CH340/CP2102 的 TX |

## 电源

| 传感器 | VCC | GND |
|--------|-----|-----|
| DHT11 | 3.3V | GND |
| MQ-2 | **5V**（模块需要加热） | GND |
| BH1750 | 3.3V | GND |
| OLED | 3.3V | GND |

注意：MQ-2 必须用 5V 供电才能正常加热并检测气体。模拟输出 AOUT 在 0-5V 范围，STM32 的 ADC 参考 3.3V，建议用电阻分压或确认模块有板上分压。

## 串口 USB 转接

```
STM32 PA9 (TX) → CH340 RXD
STM32 PA10 (RX) ← CH340 TXD
CH340 GND ←→ STM32 GND
```

## BH1750 I2C 地址

默认地址 `0x23`（ADDR 引脚接 GND）。如果需要多个设备，改接 SDA 获得地址 `0x5C`。

## 注意事项

1. **PA0-PA4 已被 OLED 占用**，不要将这些引脚用于其他外设
2. **PB6/PB7 是 I2C1**，与 USART1/2 的 REMAP 引脚重叠，使用时注意 JTAG/SWD 冲突
3. **MQ-2 需要预热**：上电后需要 ~20 秒才能稳定读数
4. **DHT11 读取超时**：单次读取最大 ~5ms，主循环中每 3 秒读一次可接受
